## What the segment actually does

Two definitions:
1. An agent as observation + internal state + action (the standard cybernetic picture)
2. The information-loss-boundary as *constitutive* scope — not an idealization but a carving of what AAD addresses

The second is the load-bearing philosophical move. By framing the observation channel's lossiness as a scope condition rather than an assumption, the theory sidesteps the question "what if the agent COULD access full environment state?" — the answer is simply "then we're outside AAD scope." Clean.

## Wandering thoughts

The "agent-environment boundary is constitutive" move reminds me of how thermodynamic systems are defined — a system is constituted by what you choose to call the system boundary. That's a deep choice that determines what counts as internal vs. external entropy. AAD makes the same move: the theory is constituted by the boundary choice.

There's something interesting about the information-loss condition being the scope condition. It means the theory is fundamentally a theory of *epistemic limitation* — of bounded perception. An agent with full access to $\Omega_t$ has no use for this theory. I wonder if the framework's claim about "logozoetic agents" (morally weighted persistence) is partly grounded in this epistemic limitation — the agent can't know its own future state, which is why continuity becomes precious.

The three-condition agent definition (perception + memory + action) will show up repeatedly. But notably, this definition doesn't yet distinguish *adaptive* from non-adaptive agents — a thermostat satisfies all three conditions. The adaptive part must come in later scope conditions.

How valuable: foundational but not surprising. 3/10 for surprise, 9/10 for load-bearing.
