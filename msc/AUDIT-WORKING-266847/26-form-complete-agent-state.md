# Reflection: form-complete-agent-state

## What the segment does

Lifts the agent state from M_t (epistemic) to X_t = (M_t, G_t) (epistemic + purposeful). This is a representational commitment, not a theorem — motivated by backward compatibility, different dynamics, and directedness of separation. Section I is the special case G_t = ∅.

## The backward compatibility argument

The most important property: Section I machinery (mismatch, gain, tempo, persistence) applies to M_t unchanged. The lift doesn't modify anything already established — it adds structure alongside M_t. This is well-engineered: the theory's Section I results don't become contingent on Section II's additional structure.

## The conjecture about uniqueness

"We conjecture that any alternative decomposition... if it preserves directed separation... will be structurally isomorphic to (M_t, G_t)." This is explicitly marked as unproved. The practical claim is: directed separation forces a component whose update function doesn't reference purpose — which uniquely picks out M_t. Whether this is provable or just a structural intuition is open.
