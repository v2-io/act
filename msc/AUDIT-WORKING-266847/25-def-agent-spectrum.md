# Reflection: def-agent-spectrum

## What the segment does

Defines the agent spectrum as a 2×2 continuum along model richness × objective richness axes: reactive / adaptive tracker / blind pursuer / actuated agent. This is definitional infrastructure for Section II — locating where AAD's machinery becomes relevant and where it runs out.

## The terminological commitment

"Actuated agent" (not "purposeful" or "goal-oriented") for the top-right quadrant. "Self-actuated" is reserved for agents that set their own objectives. These are deliberate word choices that avoid natural-language baggage.

## Naming targets surfaced

The tracker should have entries for "agent spectrum" and the individual quadrant names. "Blind pursuer" is an interesting term — pursuing a goal without modeling reality (gradient follower, basic search). "Actuated agent" is the formal term for the full top-right quadrant.

## The Miller one-state → two-state threshold

The Moore machine instantiation is valuable. One-state machines cannot cooperate or coordinate regardless of payoff structure. Two-state machines can implement Tit-For-Tat and similar strategies. This is AAD's minimum meaningful agent: the two-state machine is the minimal adaptive tracker / blind pursuer. The phrase "quantum leap into the adaptive tracker / blind pursuer boundary" is evocative.

This connects Miller's empirical finding directly to AAD's scope condition: the meaningful threshold is not "does M_t exist?" but "is M_t rich enough to support adaptive dynamics?"

## The Hafez bridge

The comparison with Hafez's agency/intelligence hierarchy is precise:
- Hafez agency ≈ AAD scope condition (scope-agency)
- Hafez intelligence ≈ AAD Section I + II machinery (learning, self-monitoring, adaptation)
- Hafez P (bi-predictability) characterizes information structure of coupling (scale-invariant)
- AAD T (adaptive tempo) characterizes corrective capacity (not scale-invariant)

They measure complementary aspects. This is a clean adoption of Hafez's framework within AAD.
